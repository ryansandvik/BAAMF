# BAAMF v1.1 — TestFlight

Thanks for testing! Here's what's new.

---

## ✨ New

- **Attendance RSVP** — Mark yourself going or not going on any month card. Tap the green pill or 👥 icon to see the full roster with profile pictures, sorted by going → no response → not going.
- **Profile pictures** — Member avatars now appear throughout the app, including the attendance roster and member lists.
- **User roles** — Virtual user roll added. Virtual users are ineligible for selection when automatically populating a schedule. Admins can still assign them a month.  
- **Phase deadlines** — Hosts can set a deadline when advancing to each phase. A countdown shows on the home card, you'll get a push notification 1 hour out, and the phase auto-advances when time's up.
- **Admin: deadline defaults** — Admins can set default phase durations under Profile → Phase Deadlines so the date picker pre-fills automatically.

## 🐛 Fixed

- Push notification sent to you when another member "Read It" vetoes your book.
- New members now appear in the app immediately without anyone needing to re-login.
- Month management controls (gear icon, configure submissions) are now only visible to the host of *that specific month*, not all hosts.
- Your own submission now shows "Your Pick" in the veto view instead of a veto button.
